# YGO Combo Maker - ユーザー運用マニュアル

「YGO Combo Maker」は、キーボードでいちいち文字を打つことなく、使いたいカードとアイコンをパレットからクリックしていくだけで流れるように「遊戯王の展開ルート」を作れる専用ツールです。

## 1. キャンバスの基本操作：展開ルートの作り方
1. 画面右側のタブから **「デッキ」タブ** を選びます。（あらかじめデッキにカードを追加しておく必要があります。手段は【2.】に後述）
2. デッキに入ったカードの一覧の中から、初動にしたいカードを **1回クリック** します。
3. カードが左側の黒い画面（キャンバス）へと即座にポンッと追加されます。
4. 次に右端の **「アクション」タブ** を開き、「通常召喚」や「効果発動」「サーチ」などの専用アイコンをクリックします。
5. 同様にして、左画面に「エクセル」→「通常召喚」→「サーチ」…と繋がるようにポンポンとクリックしていくだけで美しい展開ルートが完成します。
6. キャンバス上のカードやアイコンの順番を間違えた場合は、キャンバス上の該当アイコンをもう一度直接クリックするだけでサクッと『削除』できます。

---

## 2. デッキへのカード追加方法（3つの最強アプローチ）
本アプリでは、目的のカードを探してパレットに追加するために、3つの異なるアプローチを用意しています。環境に合わせて使い分けてください。

### A. 「追加検索」タブから1枚ずつ探して追加する
もっとも古典的な手法です。カード名を途中まで入力して検索ボタン（虫眼鏡）を押します。「すねーくあい」のような『ひらがな入力』や、「ブルーアイズ ドラゴン」のような『複数単語でのスペース区切り検索』にも対応しています。出てきたカード画像を1回クリックすると、自分が今開いているデッキへ送信・保存されます。

### B. テキストのインポート機能で「サイトのレシピ」から一括追加する
デッキタブの上部にある **青いインポートボタン（ファイル読み込みアイコン）** を押します。
マスターデュエルをまとめた攻略サイトなどのデッキレシピにあるテキストリスト（例：「3 灰流うらら」「x2 増殖するG」など）をコピーして、そのままテキストボックスへ貼り付けて「実行」を押してください。
先頭についている不要な「枚数」などのテキストをAIが自動的に無視・削除し、純粋なカード部分だけを一瞬で抽出し、全カードをデッキに構築します。

### C. 「画像切り抜き（スライサー）」機能で、画面のスクショから直で取り込む（★最強機能）
テキストすら打たずにデッキを作る魔法の機能です。
1. デッキタブにある **「画像から切り抜き」ボタン** を押します。
2. 左記の画面を開いたまま、遊戯王マスターデュエル・または攻略動画のデッキ編成画面などの**スクリーンショット（画像）を直接 `Ctrl+V` キーで画面内にペースト**して貼り付けます。
3. 画面内メニューの **「自動一括カット」** を選びます。
4. 横（列）と縦（行）のマスの欄に、貼り付けた画像のカードの並び枚数（メインデッキなら横10枚・縦4枚などの比率）を入力します。
5. あなたが貼り付けた画像上で、**「カードの集まりのカタマリ」全体をマウスでビーッと大きく四角くドラッグして囲みます**。
6. カードのマス目に沿って黄色い補助線がバッチリ引かれます。そのまま「一括カットして追加」を押してください。
7. 【ドン！】という音は出ませんが、一瞬にして40枚のカードが等分に切り出され、オリジナルのカード画像としてデッキに完璧に登録完了されます！

（※切り抜き時に生じた「空欄の切り抜きマス等」は、左側のパレットに生成された不要な画像を直接『右クリック』すれば、サクっと即座に消去できます）

## 3. 複数デッキの管理について
デッキタブの上部にあるセレクトボックスと、 `[+]`（新規） `[ペン]`（名前変更） `[ゴミ箱]`（削除） の各ボタンを使って、複数パターンのデッキを自由に管理できます。

作成中の展開コンボを崩すことなく、「スネークアイ用」「炎王用」「勇者用」とパーツごとに複数のデッキ（画像データ含む）を別々に保存していつでも使い分けることが可能です。
