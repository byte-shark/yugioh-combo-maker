# YGO Combo Maker - 完成報告書 (Final Walkthrough)

## 🎯 プロジェクトの完了
テキスト入力を一切使わず、直感的なクリック操作のみで「遊戯王マスターデュエルの展開ルート」を作成できる静的Webアプリケーションの全機能実装を完了いたしました。

### ✨ 実装・搭載された主要機能
* **公式API連携と検索**: YGOPRODeck APIから日本語カード13,000枚以上を取得・キャッシュし、高速に検索パレットを構築。ひらがなやスペース区切りでの柔軟なテキスト検索にも対応。
* **盤面構築システム**: パレットからカードや行動アイコン（召喚、サーチ等）をクリックするだけで手軽にキャンバスへ配置。右クリックやダイレクトクリックでの消去など、無駄のない洗練されたUXを達成。
* **マルチデッキ管理**: 構築中のコンボを崩さずに、パーツを保存・切り替えできる複数デッキのプロファイル機能と自動ローカルセーブ。
* **テキストインポート**: 他サイトのテキストレシピ（「3 灰流うらら」等）をペーストし、高度な正規表現で純粋なカード名だけをパースしてデッキに一括追加するロジック。
* **画像一括切り抜きスライサー（究極機能）**: マスターデュエルのデッキ画像を貼り付け、直感的なドラッグ指定と設定グリッドに基づいて自動で40〜60枚のカードへ個別に分割。一瞬で画像カード群としてデッキパレットに生成される、APIの限界を超えた革新的な機能。

### 🐛 解決された難解なCSSバグ
* **ChromeにおけるFlexbox無限伸長バグの根絶**:
  開発中に発生した、「スクロールバーが画面外に押し出されてスクロールできない」などのFlexbox特有の不治のバグに対し、大元のコンテナである `#app` に `100vh` を厳格に適用し、パレットエリアを `position: absolute` （絶対座標）によるCSSレイアウトで制御することで、どのような巨大解像度画像や大量カード描画時でも1ドットたりとも画面外に突き抜けない無敵のレイアウトを実現しました。
