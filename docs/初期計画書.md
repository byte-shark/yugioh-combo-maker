# 遊戯王MD 展開ルートエディター 初期実装計画

遊戯王マスターデュエル（MD）の展開ルートを、言語（テキスト）に頼らず直感的な「アイコン」と「カード画像」のつながりだけで誰でもスピーディーに作成できるWebアプリケーションを構築します。

## 💡 提案と技術スタック
1. **API連携・画像・リミットレギュレーションの取得**:
   遊戯王の「公式データベース」は、プログラムから直接画像を引っ張ろうとするとCORS（セキュリティ制限）で弾かれるため、そのままではブラウザアプリから読み込むことができません。
   そのため、遊戯王アプリ開発における世界標準のオープンAPIである **「YGOPRODeck API」の日本語検索対応版** の利用を提案します。
   - **メリット**: 公式と同じカード画像が高画質で取得でき、さらに**「マスターデュエルの最新リミットレギュレーション」**のデータを完備しているため、禁止・制限カードの超過判定が自動で行えます。
2. **ノーテキスト展開ルートUI (キャンバス)**:
   カードとアクションを繋げるフローチャート形式を採用します。以下のような視覚的ルートを数クリックでポチポチと生成できるようにします。
   例： `[黒魔女ディアベルスター(画像)]` ➔ `[✨効果発動アイコン]` ➔ `[原罪宝－スネークアイ(画像)]` ➔ `[🪦墓地へ送るアイコン]`
   言語に依存せず、動画を見るよりも一瞬で展開や最終盤面を視認できるルート図を作成します。
3. **完全ローカル保存**:
   パスワードマネージャーと同様のノウハウを活かし、作成した複数の展開ルートの保存や読み込みはブラウザ（`localStorage`）内で行うため、アカウント登録などの手間がなく爆速で動作します。

## ニューロン連携と最新カードについて（API調査結果）
公式アプリ「遊戯王ニューロン」および「公式データベース（コナミ）」は、セキュリティ（CORS制限等）により外部のWebアプリから自動でデータを引っ張ることを完全に遮断しています。そのため、現在世界中で個人開発されている遊戯王シミュレーターやデッキツールはすべて「YGOPRODeck」などの有志によるグローバルAPIを使用しています。

- **根本的な課題**: YGOPRODeckは海外ベースであるため、OCG（日本先行発売）の最新カード（当時最新だったスネークアイ等）の日本語翻訳データが数週間〜数ヶ月遅れる仕組みになっています。
- **本アプリでの解決策**: この制約により「新しいカードが使えない」という事態を防ぐため、検索で見つからない場合は**「1クリックで名前を入力し、疑似カードとして作成する（カスタムカード機能）」**を備えています。これにより、APIの更新やネット環境を一切気にせず、全テーマの展開ルート作成が可能です。
（※注記：この後、画像の切り抜きによる「マスターデュエルスライサー」機能へと発展しました）

## 作業ディレクトリ構造

### Frontend Structure
#### index.html
サイドバー（カード検索・デッキやアクションのパレット）と、右側の広大なメインキャンバス（ルート描画エリア）を持つSPA構造。
#### style.css
モダンでゲームライクな洗練されたデザイン。ドラッグ＆ドロップなどのインタラクションを支えるCSSアニメーションも実装。
#### app.js
APIからのカードデータ取得・キャッシュ、MD制限チェックロジック、およびキャンバス上でのカードやアイコンの追加・連結・保存を行うコアロジック。
