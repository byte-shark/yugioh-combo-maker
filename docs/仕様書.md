# YGO Combo Maker - 仕様書 (Requirement Specifications)

## 1. 概要
本アプリケーションは、遊戯王マスターデュエルのプレイヤー向けに、テキストを入力する手間を完全に省き、直感的なマウスクリック操作のみで「展開ルート（盤面コンボ）」を作成・保存できる静的Webアプリケーションである。

## 2. システム要件
* **動作環境**: 最新のモダンブラウザ (Google Chrome, Edge, Safari, Firefoxなど)
* **ネットワーク**: 初期起動時のみ公式カードデータベース通信のためのネットワークが必要。画像切り抜き機能や独自のカスタムカード等を利用すれば、オフライン状態でも主要な動作が可能。
* **バックエンド**: なし (100%クライアントサイドのWeb技術のみで完結)
* **データ保存**: ブラウザの `LocalStorage` 領域 (5MBのブラウザ制限内に最適化されたデータ永続化)

## 3. コア機能要件
1. **公式カードデータの検索**: YGOPRODeck APIを用いたカード検索機能。「ひらがな入力」や「複数単語のスペース区切り」を用いた高度なマッチングロジック。
2. **直感的な盤面構築**: パレットからカードや展開アクション（通常召喚、効果発動など）をワンクリックし、視覚的なキャンバスに並べる機能。
3. **複数デッキのプロファイル管理**: 複数のデッキ（カテゴリ）を作成・保存・ワンタッチで切り替える機能。
4. **デッキテキストのインポート**: 公式やまとめサイトなどのテキスト形式レシピ（「3 灰流うらら」や英数字の混在等）をペーストし、正規表現で純粋なカード名だけをパースしてデッキに一括追加する機能。
5. **画像一括切り抜き (スライサー)**: マスターデュエルなどのゲーム画面スクリーンショットをクリップボード等から貼り付け、指定したグリッド（列・行寸法）に基づいて自動で最大60枚のパーツに一瞬で個別・画像カードとしてデッキパレットに生成・登録する機能。
6. **未知カードの自動救済処置**: APIにまだ翻訳データが存在しない最新カードでも、ユーザーが望む文字や切り抜き画像から自動的に「即席カード」として構築され、ユーザーフローを決して止めない自己完結仕様。
